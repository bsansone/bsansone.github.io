Sequential Program Control - Energy Cost:

Objective: Write a program that calculates the annual cost of running various appliances. For each appliance, the program will request that the user enter the cost per kilowatt - hour and the number of kilowatt - hours the appliance uses in a year.

program_info = '\nWritten by Brandon Sansone\nDate: 6/9/2016\n'
annual_cost = 0.0       # Annual usage cost per appliance
total_cost = 0.0        # Total running cost of all appliances

### Output Variable Description
print (program_info)
amount = int(input("\nEnter amount of running appliances: ")); # Ask user amount of appliances to calculate

for x in range(amount): # Loop through amount of appliances from previous input
    appliance = (input("\nPlease enter the appliance name: ")); # Appliance name, string 
    cost = float(input("Please enter the cost per KW - hr of the appliance (in cents): ")); # Cost per KW - hr, float
    usage = int(input("Please enter the annual usage (in KW - hr): ")); # Annual Usage, KW - hr
    annual_cost = cost * usage # Calculates the annual usage cost per appliance
    total_cost += annual_cost # Calculates the total running cost of all appliances

print ("\nThe total cost of the annual usage is", "${:,}".format(round(total_cost,2)), ".") # Print running total

Selection Program Control - BMI Calculator:

Objective: Write a program that will generate a person's BMI result based on the persons weight and height.

### Define Variables
program_info = '\nWritten by Brandon Sansone\nDate: 6/16/2016\n'
bmi = 0.0   # Body Mass Index (BMI)

### Program Info Output
print (program_info)

### Gather User input
weight = float(input("\nEnter your weight (pounds): ")); # Ask user for weight (in pounds)
height = float(input("Enter your height (inches): ")); # Ask user for height (in inches)

### Calculations
bmi = weight * (703/height**2)

### Conditionals
if bmi < 18.5:
    print ("\nBMI: ", "{:,}".format(round(bmi,1)))
    print ("\nBased on the BMI Calculator you are...underweight\n")
elif bmi > 25:
    print ("\nBMI: ", "{:,}".format(round(bmi,1)))
    print ("\nBased on the BMI Calculator you are...overweight\n")
else:
    print ("\nBMI: ", "{:,}".format(round(bmi,1)))
    print ("\nBased on the BMI Calculator you are...optimal weight\n")
